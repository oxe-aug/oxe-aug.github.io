OXE-Aug is a large-scale robot-augmented dataset built on the Open-X Embodiment (OXE) dataset. It augments 16 popular real-world datasets into 9 different robot embodiments using a combination of simulation-based rendering and inpainting.

Experimental Settings

We evaluate our augmentation method using the following experimental configurations:

  • Source Only: Use only the original demonstrations from the source robot.
  • Target Only: Use only the augmented demonstrations for the target robot (no source data).
  • 1+1 (Unseen): Combine original and augmented demos, where neither robot is the target robot.
  • 1+1 (Seen): Combine original and augmented demos, where the augmented robot is the target robot.
  • N-1 (Leave-One-Out): Train on all robot demonstrations except the target robot.
  • N (All Robots): Train on all robot demonstrations.
Note: Only Target Only, 1+1 (Seen), and N configurations train on the target robot's data.

Simulation Results

Scaling the number of augmented robots seems to improve policy performance, especially with generalisation to unseen robots!

We also see improvements in robustness to visual disruptions. We tested two altered environments, changing lighting conditions and partially blocking the robot from view.

Similar to previous performance gains, this robustness scales up as we include more robots in the training process.
